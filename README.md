# Polemos Workshop

La transformación deja marcas.

Algunas se convierten en cicatrices. Otras en tatuajes.

## Descripción

Polemos es una plataforma digital creada para representar el trabajo, la filosofía y el proceso detrás de mi oficio como tatuador.

Este proyecto también funciona como laboratorio de aprendizaje para reforzar conocimientos de desarrollo web moderno utilizando HTML, CSS, JavaScript, Git y tecnologías que serán incorporadas progresivamente en futuras versiones.

## Stack Inicial
